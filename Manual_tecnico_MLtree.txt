Maximum likelihood part

To get the first tree, it will be necessary the installation of some softwares:


MAFFT (Multiple Alignment using Fast Fourier Transform) is a popular alignment software for multiple sequences. 
Install MAFFT: To use MAFFT, you will first need to install it on your computer.
You can download the appropriate version for your operating system from the MAFFT website (http://mafft.cbrc.jp/alignment/software/)


Modeltest-MG is a software for model selection and assessment of evolutionary models for nucleotide and amino-acid data.
Install Modeltest-MG: To use Modeltest-MG, you will first need to install it on your computer.
You can download the appropriate version for your operating system from the Modeltest-MG website (https://github.com/ddarriba/modeltest)

RAxML is a software for phylogenetic tree inference using maximum likelihood.
Install RAxML: To use RAxML, you will first need to install it on your computer.
You can download the appropriate version for your operating system from the RAxML website (https://cme.h-its.org/exelixis/web/software/raxml/index.html)
